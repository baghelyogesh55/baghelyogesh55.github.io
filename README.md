# Yogesh Baghel — 3D Developer Portfolio

A modern, interactive portfolio for a Salesforce Developer, built with **vanilla HTML/CSS/JS** and **Three.js** (loaded via CDN — no build step).

## ✨ Features
- Interactive 3D particle-network hero, floating geometry, and orbiting certification scene (Three.js r160)
- Custom **spaceship cursor** with thruster trail and direction-aware rotation
- Typed hero intro, animated counters, scroll-reveal, animated skill bars
- Dark / light theme toggle (persists), mobile nav, back-to-top, contact form
- Fully responsive, accessible (`prefers-reduced-motion`), SEO + Open Graph meta

## 🚀 Run locally
No build needed. Use any static server:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```
